## 三、参考书籍：
### 推荐：
目前，最好的Java相关技术路线地图是《Java工程师成神之路》，我个人以为按图索骥，照着学习就可以得到一个完整的Java技术体系。
对个人而言，相对完整的知识技术体系的意义，如同完整的工业体系对中国的意义一样，在当今大争之世中，这是一种最能保持独立自主的能力，也是一种能主宰自己命运的力实力。
